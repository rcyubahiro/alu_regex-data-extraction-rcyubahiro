# Regex Data Extraction Project

## Overview
This project uses Python and Regular Expressions to extract structured data (like emails, URLs, phone numbers, etc.) from text.

## Extracted Data Types
- Email addresses
- URLs
- Phone numbers
- Credit card numbers
- Time (12h and 24h)
- HTML tags
- Hashtags
- Currency amounts

## Technologies Used

- Python 3
- Regex (re module)
